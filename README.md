

## Dev config

Uses jekyll 
- [install](https://jekyllrb.com/docs/installation/)
- [quickstart](https://jekyllrb.com/docs/)

- Install Ruby >= 2.5.0

```sh
bundle install
bundle exec jekyll serve
```

Browse http://localhost:4000

To add new pages:
- add .md file to _pages folder
- add title, url to _data/navigation.yml
